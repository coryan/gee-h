# gee-h

A C++14 client library for etcd leader election.

## Install

This library is built using `cmake(1)`.  On Linux and other Unix variants the usual commands to build it are:

```commandline
cmake .
make
make install
```

On Windows the commands are

```commandline
cmake .

```
